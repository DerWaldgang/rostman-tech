# WithdrawApp

Интерфейс управления выводом USDT на Next.js 16 App Router.

## Запуск

```bash
npm install
npm run dev
```

**Тестовые данные для входа:** `admin` / `password`

---

## Страницы

`/` | Главная — навигационные карточки |
`/login` | Авторизация |
`/withdraw` | Создание вывода |
`/withdrawals` | Поиск вывода по ID |
